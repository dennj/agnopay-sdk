# Changelog

All notable changes to the AgnoPay SDK will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of AgnoPay SDK
- Client-side order creation with publishable keys
- React hooks (`useAgnoPayCheckout`)
- React components (`AgnoPayCheckout`)
- TypeScript support with full type definitions
- Configurable API and wallet URLs
- Customizable checkout styling
- Server-side utilities for Next.js
- ESM and CommonJS support
- Tree-shaking support

## [0.1.0] - 2024-10-30

### Added
- Initial alpha release
- Core functionality for order creation
- Basic React integration
- TypeScript types
